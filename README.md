# color-cycle

  A test of the component tool from TJ Holowaychuk.

## Installation

  $ component install mvolkmann/color-cycle

## API

  ```javascript
  var cc = require('color-cycle');

  // Get component HTML and appends to document.
  $('body').append($(cc.getHtml()));

  // Set up event handling.
  cc.setup();
  ```
